> College Bus Tracking Mobile App — Figma UI Design

A clean, modern mobile UI for students to track college buses in real time.
Built as a high-fidelity Figma design with 7 screens at 390×844px.

---

 Screens

| # | Screen | Description |
|---|--------|-------------|
| 01 | Splash | Logo, tagline, Get Started CTA |
| 02 | Login | College email/password, Google SSO |
| 03 | Home Dashboard | Live bus card, ETA, route preview |
| 04 | Live Tracking | Map view, stop timeline, seat count |
| 05 | Routes | Searchable route list with status badges |
| 06 | Notifications | Delay alerts, arrival and route change cards |
| 07 | Profile | College ID card, saved routes, settings |

---

 Design Specs

- **Frame Size:** 390 × 844 px (iPhone 14)
- **Font:** Poppins (Google Fonts)
- **Corner Radius:** 16px cards · 12px inputs · 100px badges
- **Base Spacing Unit:** 8px grid

Figma Import

1. Unzip `BusMate_Figma_Screens.zip`
2. In Figma → **File → Import** → select all 7 SVGs
3. Each SVG becomes a 390×844 editable frame
4. Register colour tokens as **Local Colour Styles**
5. Wire Bottom Nav items using the **Prototype** tab
