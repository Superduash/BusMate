# Overview 👋

BusMate is a modern college bus tracking mobile application UI designed to improve the daily transportation experience for students.

The application helps students track buses in real time, view estimated arrival timings, receive transport alerts, browse routes, and manage schedules through a clean and minimal mobile interface.

Designed using Figma, BusMate follows modern UI/UX principles focused on:
- Simplicity
- Readability
- Smooth navigation
- Responsive layouts
- Real-world usability

---

# Features ✨

- 🚌 **Live Bus Tracking**
  - Track buses with real-time route visualization.

- ⏱️ **ETA & Arrival Updates**
  - Displays estimated arrival timing and stop information.

- 🔔 **Transport Notifications**
  - Receive alerts for delays, arrivals, and route changes.

- 🗺️ **Route Management**
  - Browse available routes, stops, and boarding points.

- 📅 **Schedule Dashboard**
  - View pickup and drop schedules easily.

- 👤 **Student Profile**
  - Access transport details and saved routes.

- 📱 **Modern Mobile UI**
  - Card-based layouts with minimal design philosophy.

- 🎨 **Purposeful Color System**
  - Green → On Time  
  - Orange → Delayed  
  - Blue → Primary Actions  
  - Red → Alerts

# UI Screens 📱

The BusMate case study contains multiple high-fidelity mobile screens:

## 🔐 Login Screen
Clean authentication interface for student access.

## 🏠 Home Dashboard
Displays live bus information, ETA, routes, and quick actions.

## 📍 Live Tracking
Shows current bus movement, stops, and journey progress.

## 🛣️ Routes Screen
Displays route details and boarding points.

## 🔔 Notifications
Provides delay alerts and important transport updates.

## 👤 Profile Screen
Displays student details, saved routes, and settings.

## 📆 Schedule Screen
Shows daily pickup and drop timings.

- Clean and readable mobile typography

- Rounded modern components
- Bottom navigation for faster access
